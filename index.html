Welcome to WPDK.

WPDK (WordPress Development Kit) is an awesone Open Source framework to develop in WordPress.

Requirements

  • Wordpress 3.4 or higher (last version is suggest)
  • PHP version 5.2.4 or greater
  • MySQL version 5.0 or greater
