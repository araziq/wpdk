Welcome to WPDK.

WPDK (WordPress Development Kit) is an awesome Open Source framework to develop in WordPress.

Requirements

  • Wordpress 3.4 or higher (last version suggested)
  • PHP version 5.2.4 or greater
  • MySQL version 5.0 or greater

Getting Started

  1. Copy wpdk folder in your plugin directory.
  2. Include the main wpdk.php
  3. Enjoy

Organization of the file system

The file system structure is not binding: following this standard nomenclature and organization is strongly recommended, in order to make it readable and compliant with other plugins.

Here it is:

your_plugin_directory/

  • wpdk/
  • index.php
  • (main).php

Folders

wpdk

(main).php

This is the main file of the plugin and you can name it as you like. In this file you have to:

  • Set WordPress standard comments
  • Include WPDK framework
  • Start your plugin
